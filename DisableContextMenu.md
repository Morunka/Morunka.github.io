# Отключение контекстного меню в играх

Этот репозиторий содержит решения для отключения контекстного меню (правого клика) в играх, особенно на чёрных полях по бокам.

## Решения

### 1. JavaScript-скрипт (disable-context-menu.js)

Этот скрипт можно добавить в вашу игру, если у вас есть доступ к исходному коду.

```html
<script src="disable-context-menu.js"></script>
```

### 2. Обёртка в iframe (disable-context-menu-iframe.html)

Если у вас нет доступа к исходному коду игры, вы можете использовать этот HTML-файл как обёртку для вашей игры.

1. Откройте файл `disable-context-menu-iframe.html`
2. Замените `URL_ВАШЕЙ_ИГРЫ` на реальный URL вашей игры
3. Загрузите этот файл на хостинг вместо оригинальной игры

## Для игр на Scratch или Kogama

Если ваша игра создана на Scratch или Kogama, вы можете добавить следующий код в начало вашего проекта:

```javascript
// Отключаем контекстное меню
document.addEventListener('contextmenu', function(event) {
    event.preventDefault();
    return false;
});
```

## Для игр на Яндекс.Игры

Если вы публикуете игру на Яндекс.Игры, вы можете:

1. Добавить скрипт `disable-context-menu.js` в вашу игру
2. Или использовать обёртку `disable-context-menu-iframe.html` и загрузить её вместо оригинальной игры

## Примечания

- Некоторые браузеры могут блокировать отключение контекстного меню из соображений безопасности
- Для iframe может потребоваться настройка CORS-политики, если игра загружается с другого домена
- Если игра использует Canvas или WebGL, может потребоваться дополнительная настройка
